## The template

A simple React + Tailwind CSS template with TypeScript, it contains the base configurations of tailwind and a theme switcher example using the lib @headlessui for complexes UI components. 

## Instructions

```bash
# install 
npm install
# run on development
npm run dev
# deploy 
npm run deploy
```
